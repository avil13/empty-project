# empty-project
quick start developing a simple front-end application

In project directory:
```
npm install
```

Now now you have npm commands:

- `npm start` - Run web server
- `ng` - angular-cli component `npm run ng g c page/new-page`
- `lint` - linting
- `test` - run test
- `pree2e` - run test
- `e2e` - run test
- `build` - build project
